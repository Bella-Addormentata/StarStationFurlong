# IDEAS Tech Tree

* Core Progression
  * Players unlock skills through repeated use, station contribution, and successful missions.
  * Each skill rank improves efficiency, unlocks new recipes, and expands the player’s role in the colony.
  * Skill advancement should feel like a practical career path rather than a pure combat stat grind.

* Skill Categories
  * Survival
    * Food production, repair, medical care, and basic habitat upkeep.
  * Engineering
    * Ship maintenance, station infrastructure, power systems, and machine operation.
  * Trade
    * Pricing, market analysis, cargo handling, and inventory management.
  * Crafting
    * Furniture, tools, machine parts, and station modules.
  * Exploration
    * Navigation, mapping, scouting, and outpost deployment.

* Advancement Model
  * Skills gain experience through real actions such as crafting, fixing, hauling, and operating systems.
  * Higher ranks unlock better tools, stronger recipes, and access to more advanced stations or jobs.
  * A small number of key skill slots should define a player’s specialization, like a career path in a simulation game.

* Colony Impact
  * Skill growth should directly improve the station’s output and survival odds.
  * Skilled players can improve outpost efficiency, reduce waste, and increase production of critical supplies.
  * Advanced roles should support team play, cooperative building, and shared progression.

* Long-Term Goals
  * Players can eventually specialize in station management, trade routes, shipbuilding, or frontier leadership.
  * Skill trees should connect to crafting and market systems so every advancement has a clear gameplay use.
  * Progression should reward both expertise and contribution to the wider civilization.
